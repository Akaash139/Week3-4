# 📊 Week 03 & 04 – Algorithm Implementation (Java)

---

## 📌 Overview

This project contains solutions to **Week 03 & Week 04 problem statements**, focusing on the implementation of **sorting and searching algorithms** in Java.

The problems are designed around **real-world financial and data-processing scenarios**, helping bridge the gap between theoretical concepts and practical applications.

---

## 🎯 Objectives

* Implement fundamental **sorting algorithms** (Bubble, Insertion, Merge, Quick)
* Apply **searching techniques** (Linear Search, Binary Search)
* Understand **time and space complexity**
* Solve **real-world problems using algorithms**
* Improve problem-solving and coding skills

---

## 🧠 Concepts Covered

### 🔹 Sorting Algorithms

* Bubble Sort (O(n²))
* Insertion Sort (O(n²), adaptive)
* Merge Sort (O(n log n), stable)
* Quick Sort (O(n log n) average)

### 🔹 Searching Algorithms

* Linear Search (O(n))
* Binary Search (O(log n))

### 🔹 Additional Concepts

* Stability in sorting
* Comparator usage
* Time complexity analysis
* Handling duplicates
* Key–value data modeling

## 🚀 Problem Descriptions

### ✅ Problem 1: Transaction Fee Sorting

* Bubble Sort for small datasets
* Insertion Sort for medium datasets
* Stable sorting with duplicate handling
* Detect high-fee outliers

---

### ✅ Problem 2: Client Risk Score Ranking

* Sort clients by risk score
* Ascending (Bubble Sort)
* Descending (Insertion Sort)
* Identify top high-risk clients

---

### ✅ Problem 3: Trade Volume Analysis

* Merge Sort (ascending)
* Quick Sort (descending)
* Merge sorted lists
* Compute total trade volume

---

### ✅ Problem 4: Portfolio Return Sorting

* Sort assets by return rate
* Handle tie conditions
* Apply custom sorting logic

---

### ✅ Problem 5: Account ID Lookup

* Linear Search (unsorted data)
* Binary Search (sorted data)
* Handle duplicates
* Count occurrences

---

### ✅ Problem 6: Risk Threshold Lookup

* Binary search variations
* Find floor and ceiling values
* Determine insertion position
